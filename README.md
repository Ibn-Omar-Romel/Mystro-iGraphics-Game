<h2>📝 Project Overview</h2>
<p align="justify">
  <b>MYSTRO</b> is a custom-built 2D desktop game developed from scratch using <b>C++</b> and the <b>iGraphics</b> library. This project demonstrates foundational game development mechanics including game loops, sprite-based animation, collision detection, and interactive graphical user interfaces.
</p>

<h2>🌟 Core Features</h2>
<ul>
  <li><b>Custom Sprites & Animation:</b> Fluid character movement and combat mechanics using sequential sprite sheets (e.g., Sword Character animations).</li>
  <li><b>Interactive UI:</b> Custom BMP-based buttons for main menus, controls, and game navigation.</li>
  <li><b>Audio Integration:</b> Background music and localized sound effects mapped to specific game events.</li>
  <li><b>Dynamic Environments:</b> Multi-level backgrounds handling different stages of gameplay.</li>
</ul>
